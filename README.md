### **Test Plan for demoblaze.com** 
________________________________________
&nbsp;

#### 1. **Objective**  <br> 
<h6> Verify that the core shopping features on demoblaze.com function correctly for a standard user  -- including login, product browsing, add to cart, and checkout. The goal is to ensure users can complete a purchase without errors. 
&nbsp;
 
 #### 2. **Scope** - what will be tested <br>
<h6> - User login & logout <br> - Product browsing <br> - Add to cart <br> - Remove from cart <br> - Checkout form <br> - Navigation links <br> - Responsive layout <br> - Form validation
&nbsp;
 
#### 3. **Out of scope** - will NOT be tested <br>
<h6> - Payment processing <br> - Admin panel <br> - Email notifications <br> - Performance testing <br> - Security / penetration 

&nbsp;
________________________________________


&nbsp;
#### 4. **Test types** <br>
<h6> - Functional    UI / visual <br> 
 - Negative testing  ·  Usability <br> 
 - Cross-browser (Chrome, Firefox) <br> 

 
 #### 5. **Test environment** <br> 
<h6> - URL: demoblaze.com <br> 
<font size="2"> - Browser: Chrome 124, Firefox 125 <br> 
<font size="2"> - Device: Desktop + Mobile (375px) <br> 


#### 6. **Tools** <br>
<h6> - Test cases: Google Sheets <br> 
<font size="2"> - Bug tracking: Jira (free tier) <br> 
<font size="2"> - Screenshots: Snagit / Lightshot <br> 

 #### 7. **Timeline** <br> 
<h6> - Test case writing: Day 1 <br> 
<font size="2"> - Test execution: Day 2 <br> 
<font size="2"> - Bug reporting + summary: Day 3 <br> 

&nbsp;

________________________________________

&nbsp;

#### 8. **Entry & exit criteria** <br>

<h6> ** Start testing when... ** <br>
<font size="2"> - Site is accessible <br>  - Test cases are written <br>  - Test account is created on demoblaze
<br>
<font size="2">    **Stop testing when...** <br>
<font size="2"> - All 20 test cases executed <br> - All bugs logged <br> - Summary report written
&nbsp;

#### 9. **Risks & assumptions** <br>
<h6> - Demo site may be unstable or reset periodically <br>   -  No real payment data will be used <br> -  Testing is manual only (no automation) <br>   -  Site data may not persist between sessions
